## Usage

### Define module

#### UMD
```html
<script src="path/to/dist/reactions.min.js"></script>
<script>window.customElements.define('reaction-list', window.WCReactions.ReactionList);</script>
```

### In HTML

### Generic usage
```html
<reaction-list></reaction-list>
```

## How to

### Demo
```sh
npm run demo
```
